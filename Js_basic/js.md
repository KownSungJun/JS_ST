# Js 연결 방법
1. `<script>`를 html에 넣어서
2. `<script src="">` (이때 `<body>`보다 아래에 해야 html 랜더된 이후에 접근 가능해짐)

# Selector(선택자)
1. tag name Selector `document.getElementsByTagName(태그명)[순서];`
2. class Selector `document.getElementsByClassName(클래스명)[순서];`
3. id Selector `document.getElementById(아이디명)`
4. `document.querySelector('div');` (이때 모든 형제가 선택됨)
5. `document.querySelectorAll('div')[순서];`
